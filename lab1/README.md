# Lab 1 – Introduction to Python for AI

## Overview

This lab focuses on fundamental Python libraries used in Artificial Intelligence, including NumPy, Matplotlib, and Pandas. It consists of multiple tasks designed to build basic data handling and visualization skills.

## Tasks Included

### Task 1: Importing NumPy

* Imported NumPy library for numerical computations.

### Task 2: Generating Even Numbers

* Generated an array of even numbers using NumPy (`arange`).
* Range: 200 to 240

### Task 3: Random Number Generation

* Generated random numbers using normal distribution.
* Used mean (μ = 0) and standard deviation (σ = 1)

### Task 4: Data Visualization

* Plotted:

  * Cosine function
  * Linear function (2x + 1)
* Used Matplotlib for graph plotting

### Task 5: Pandas Series Conversion

* Created a Pandas Series with mixed data types
* Converted values to numeric format
* Handled non-numeric values using `NaN`

### Task 6: DataFrame Creation

* Created a DataFrame using dictionary data
* Displayed structured tabular data
