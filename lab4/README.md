# Lab 4 – Informed Search / Heuristic-Based Algorithms

## Overview

This lab focuses on informed search techniques in Artificial Intelligence. Unlike uninformed search (BFS/DFS), these algorithms use heuristics to guide the search process more efficiently toward the goal.
